## Pasos a seguir para ejecutar el proyecto

Para probar la aplicacion localmente, sigue estos pasos:

### 1. Instalar las dependencias
Necesitamos instalar la herramienta 'http-server' para poder servir la web correctamente respetando la estructura de carpetas.

Para ello, ejecuta este comando en la terminal:

```bash
npm install
```

### 2. Arrancar el servidor 
Hemos configurado un comando para lanzar la web direcatamente desde la carpeta _public_ en el puerto 8080. 

Ejecuta:

```bash
nmp start
```

### 3. Abrir en el navegador
Una vez veas que el servidor ha arrancado, entra aqui:
http://localhost:8080
